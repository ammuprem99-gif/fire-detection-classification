# Fire Detection Classification

This project uses a Random Forest machine learning model to classify fire hazards based on environmental sensor data. 

## Dataset Features
* **CO & CO2**: Carbon Monoxide and Carbon Dioxide levels
* **Temperature**: Environmental temperature
* **Smog**: Smog density levels
* **Class_label**: Target classification (No Fire, May be Fire, Fire)

## Performance
* **Model Accuracy**: 97.5%
